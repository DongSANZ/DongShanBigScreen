## 📖 项目简介

**东山数据大屏**（DongShanBigScreen）是一个面向教学的数据可视化大屏项目，旨在帮助学生从零开始学习并实践数据大屏的开发全流程。

本项目将涵盖：

- 📊 数据可视化基础（图表类型、配色、布局）
- 🛠 前端大屏开发技术栈（HTML / CSS / JavaScript / ECharts）
- 🔌 数据获取与处理（API 对接、数据清洗、格式转换）
- 🎨 大屏 UI 设计思路（适配、动效、交互）
- 📦 工程化实践（Git 版本管理、模块化开发、项目部署）

## 🎯 适用对象

- 计算机相关专业学生
- 前端初学者
- 对数据可视化感兴趣的开发者

## 🗺 学习路线

| 阶段 | 内容 | 说明 |
|------|------|------|
| ① 基础篇 | 环境搭建与项目初始化 | Git、VS Code、Node.js 环境配置 |
| ② 布局篇 | 大屏页面结构与 CSS 布局 | Flex / Grid 弹性布局、等比缩放适配 |
| ③ 图表篇 | ECharts 图表组件开发 | 柱状图、折线图、饼图、地图等常用图表 |
| ④ 数据篇 | 数据获取与动态渲染 | API 对接、WebSocket 实时数据、Mock 数据 |
| ⑤ 动效篇 | 动画与交互效果 | CSS 动画、数据更新动画、交互反馈 |
| ⑥ 部署篇 | 项目打包与上线 | 静态资源部署、性能优化、Nginx 配置 |

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/DongSANZ/DongShanBigScreen.git

# 进入项目目录
cd DongShanBigScreen
```
